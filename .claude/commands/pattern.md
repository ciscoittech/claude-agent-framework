# /pattern Command

Create new agent patterns or improve existing ones.

## Usage
`/pattern "description"`

## Workflow
1. **Analysis** (30s): Load architect + researcher + performance-analyst in parallel
2. **Implementation** (45s): Load engineer + reviewer in parallel
3. **Documentation** (15s): Load documentation-specialist

- framework-senior-engineer

- documentation-specialist

## Contexts
