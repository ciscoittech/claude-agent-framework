# /validate Command

Comprehensive testing and validation of framework components.

## Usage
`/validate [component]` or `/validate --performance`

## Workflow
1. **Planning** (10s): Load validation coordinator
2. **Testing** (60s): Load multiple validators in parallel
3. **Analysis** (15s): Load results analyst

## Agents
- framework-validation-engineer
- framework-code-reviewer

## Contexts
- framework-architecture.md
- performance-optimization.md

Target: 85s total execution time.
